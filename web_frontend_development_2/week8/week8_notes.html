Ch. 8: CSS Transforms and Transitions

  • the CSS transform property lets you change things like size, positioning, and orientation of an element
  • transforms are applied in the order that they are provided
  • the translate function allows you to move an item but does not work on inline elements
  • scaling an element allows it to grow or shrink by the specified amount
  • the growing or shrinking happens from the center of the element unless you use transform-origin
  • if using more than one transform on the same element, use a space separated list, DO NOT re-declare the transform property because it will overide the previous transform
  • the rotate function will rotate by a specified number of degrees from the element's center
  • transform-origin changes the starting point of the transforms; default is from the center
  • CSS transitions provide a simple animation and can be activated by hover or by Javascript changing an element's class or state
  • transition properties are applied to the first block of code (the default code before any changes)
  • not all properties can be animated; here's a list of animatable properties
  • .2 seconds (200 milliseconds) is usually considered an ideal transition time
  • read here for some things that can be done with the transition-timing-function
  • be careful naming keyframes so that your name doesn't conflict with animation property values such as forwards, running, or alternate

Ch. 12: Canvas, SVG, and Drag and Drop

  • the width and heighth attributes must be set on the canvas element and not through css
  • the canvas coordinate system starts with (0, 0) as the top left corner
  • you can take an img element and draw it on the canvas like this:
    function drawImageToCanvas() {
    var canvas = document.getElementById("demo6");
    var context = canvas.getContext("2d");
    var image = document.getElementById("myImageElem");
    context.drawImage(image, 0, 0); }
  • the reason for drawing your image to the canvas is so that you can manipulate pixels
  • Raphael is a Javascript library for working with SVG
  • there's a draggable attribute that can be set on elements
