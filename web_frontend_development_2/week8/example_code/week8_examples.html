How to Use CSS Transforms

Translate

Hover on these boxes. The first box has negative values in its translate function. This moves the box to the left and up. The second box has positive values, so it moves to the right and down.

Scale

Hover on the boxes. The first one will grow. The second one will shrink.

Rotate

Hover on the boxes. The first one will rotate clockwise because a positive number of degrees has been provided (20deg). The second will rotate counterclockwise because a negative number of degrees has been provided (-20deg).

Skew

Hover on the boxes. The first one is set to skew(20deg, 5deg) and the second one is set to skew(-20deg, -5deg).

Transform-Origin

Hover on the boxes. They are both set to rotate by 20deg, but the first box has transform-origin: 10% 10% and the second box has transform-origin: bottom right.

Transitions

Hover on the boxes. The first one has a very slow transition to make the transition more visible. The second one has a normal transition speed of .2s.
