Ch. 9: The Window Object

  • Here is some annoying example code.
  • Javascript always has a global object; in the case of websites, that is the browser window
  • variables with global scope are actually properties of the global object
  • the reload method seems like it could come in handy; it forces the browser to completely reload the page without relying on cached info
  • you can access the browser's history and navigate back one page or forward one page like this:
    window.history.go(1); // goes forward 1 page
    window.history.go(0); // reloads the current page
    window.history.go(-1); // goes back 1 page

Ch. 14: HTML5 APIs

  • local storage allows us to save any Javascript object in local storage by saving it as a JSON object:
    localStorage.setItem("keyName", JSON.stringify(objectName));
  • we can retrieve a Javascript object from local storage like this:
    object = JSON.parse(localStorage.getItem("keyName"));
  • web workers help keep a page from freezing while some large time-consuming calculation is taking place
  • you can use system notifications if you obtain permission
  • the audio and video html tags can now be used and come with a whole bunch of built-in methods which are pretty darn neat
  • the canvas element allows javascript to draw in whatever space has been blocked out for it through the canvas' width and heighth attributes
  • shims and polyfills help people with older browsers to have an experience consistent with up-to-date browsers that do support a particular API
